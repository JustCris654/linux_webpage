Benvenuti nella home page di Arch Linux

ArchLinux is a linux distribution for computers. The ArchLinux repositories contains both free and nonfree software. ArchLinux adheres to five principles: simplicity, modernity, pragmatism, user centrality and versatily.

Package Manager

A package manager written specifically for Arch Linux, Pacman, is used to install, remove and update software packages. Also ArchLinux is a rolling-disto, that means there are no major releases of completely new version of the system but a regular system update is all you need to get the latest Arch Software.

Pros of archlinux

The strong point of an arch installation is customizability because after an arch fresh installation you have nothing except for the os and the base packages. You have to install all drivers, desktop environment, window manager, applications etc.
